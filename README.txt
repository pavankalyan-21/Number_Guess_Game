# Number Guessing Game

## Description

The Number Guessing Game is a Python-based interactive console game where players try to guess a randomly generated number between 1 and 100. The game offers three difficulty levels, each providing a different number of attempts to guess the correct number.

## Features

- **Difficulty Levels**:
  - **Easy**: 10 attempts
  - **Medium**: 5 attempts
  - **Hard**: 3 attempts
- **Feedback System**: Players receive hints whether the guessed number is too high or too low.
- **Replay Option**: Players can choose to play multiple rounds of the game.

## How to Play

1. Run the Python script in your terminal.
2. Enter your name when prompted.
3. Choose a difficulty level:
   - 1 for Easy
   - 2 for Medium
   - 3 for Hard
4. Input your guesses. The program will guide you if your guess is too high or too low.
5. Continue until you guess the correct number or run out of attempts.
6. Decide whether to play again or exit the game.

## Requirements

- Python 3.x

## Installation

1. Clone the repository or download the script:
   ```bash
   git clone https://github.com/your-username/Number-Guessing-Game.git
   cd Number-Guessing-Game
   ```
2. Ensure Python 3.x is installed on your system.

## Running the Game

Run the script using the following command:

```bash
python Number_Guessing_Game.py
```
